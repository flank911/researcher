# EXP-5 — Параллельное исполнение

- **Эпик:** E3 — Experiment Engine
- **Статус:** todo
- **Зависимости:** EXP-4

## Описание

Параллельный запуск прогонов.

- Process pool воркеров; каждый воркер пишет свой Parquet-партишн результатов.
- Агрегатор сводит `run_metrics` в каталог.
- Совместимость с dedupe/resume (EXP-3).

## Критерии приёмки (DoD)

- [ ] Ускорение близко к числу доступных ядер.
- [ ] Отсутствие гонок при записи (повтор теста EXP-4 под нагрузкой).
- [ ] Падение одного воркера не рушит весь эксперимент; resume доделывает.
