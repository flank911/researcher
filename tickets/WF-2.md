# WF-2 — WF-оркестратор (out-of-sample)

- **Эпик:** E5 — Walk-forward
- **Статус:** todo
- **Зависимости:** WF-1, EXP-2, AN-3
- **Реализует улучшение:** №8 (честная OOS-проверка, без утечки будущего)

## Описание

Walk-forward исполнение.

- На train-окне ищем лучшие параметры (по stability/целевой метрике).
- Параметры выбираются **только из прошлого**; торгуем ими на test-окне.
- Склейка test-сегментов в непрерывную out-of-sample equity.

## Критерии приёмки (DoD)

- [ ] На одном символе строится непрерывная OOS-кривая капитала.
- [ ] Тест доказывает: выбор параметров не использует данные test-окна (нет утечки).
- [ ] OOS-метрики считаются по склеенной кривой.
