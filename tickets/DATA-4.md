# DATA-4 — Каталог датасетов + dataset_hash

- **Эпик:** E1 — Data Core
- **Статус:** todo
- **Зависимости:** DATA-2, INF-3
- **Реализует улучшение:** №12 (версионирование данных)

## Описание

Каталог датасетов в SQLite + контентное версионирование.

- Таблица `datasets` (id, name, symbol, timeframe, exchange, market, start_at,
  end_at, data_path, data_hash, created_at).
- Снапшоты по `[symbol, timeframe, range]`.
- Контентный `data_hash` (нормализованный, стабильный) через INF-3.
- Политика дозагрузки: докачка создаёт новую версию датасета, не ломая ссылки
  старых прогонов.

## Критерии приёмки (DoD)

- [ ] Одинаковые данные → одинаковый `data_hash`.
- [ ] Докачка свечей создаёт новую версию; старая версия остаётся доступной.
- [ ] `DatasetRef` разрешается из каталога по id/хэшу.
- [ ] Тест на стабильность хэша.
