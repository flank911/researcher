# EXP-4 — Хранилище результатов (Parquet + каталог)

- **Эпик:** E3 — Experiment Engine
- **Статус:** todo
- **Зависимости:** EXP-2, DATA-4
- **Реализует улучшение:** №11 (запись результатов при параллелизме)

## Описание

Раздельное хранение тяжёлых артефактов и сводных метрик.

- `trades` / `equity_curve` → партиционированный Parquet (`results/`, партиции по run).
- `run_metrics` + индекс ранов (`run_hash`, статус, пути к артефактам, git commit) →
  SQLite (каталог).
- Таблицы согласно схемам из плана (разделы 3.2, 14).
- Запись логов/ошибок run.

## Критерии приёмки (DoD)

- [ ] 1000+ ранов пишутся без блокировок БД.
- [ ] Чтение по `run_hash` находит метрики и артефакты.
- [ ] Тест конкурентной записи (несколько писателей).
- [ ] Сохраняются все 11 пунктов из раздела 14 плана.
