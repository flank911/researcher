# BT-2 — Событийный движок: позиции/ордера

- **Эпик:** E2 — Backtest Core
- **Статус:** done
- **Зависимости:** BT-1
- **Реализует решение:** событийный bar-by-bar движок

## Описание

Ядро событийного бэктеста (bar-by-bar).

- `backtest/engine.py` (главный цикл по барам).
- `backtest/position.py`, `backtest/orders.py`, `backtest/portfolio.py`.
- Поддержка long/short, переворот по сигналу (`close_on_reverse_signal`),
  отслеживание открытой позиции и баланса.
- Формирование списка сделок (`trades`) и кривой капитала (`equity_curve`).

## Критерии приёмки (DoD)

- [ ] На ручном сценарии последовательность сделок и итоговый PnL совпадают
      с расчётом вручную.
- [ ] Long и short позиции открываются/закрываются корректно.
- [ ] Переворот по сигналу закрывает старую и открывает новую позицию.
