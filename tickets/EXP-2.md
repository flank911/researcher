# EXP-2 — Непрерывный прогон + нарезка по слайсам

- **Эпик:** E3 — Experiment Engine
- **Статус:** todo
- **Зависимости:** BT-7, DATA-6, EXP-1
- **Реализует улучшение:** №2 (непрерывный прогон, нарезка equity)
- **Критический путь:** да

## Описание

Эффективное исполнение research-режима.

- `experiments/orchestrator.py` + `runner.py`.
- Один полный bar-by-bar прогон на `(symbol × timeframe × params)` по всей истории.
- Сегментация equity/trades по `TimeSlice`; per-slice метрики считаются из сегмента
  **непрерывной** equity (баланс переносится между окнами).
- Опциональный режим `independent` (каждый слайс с `initial_balance`) — флагом.

## Контекст

Снижает число полных прогонов с ~25 920 до ~432 для примера из плана и
устраняет проблему прогрева индикаторов на границах окон.

## Критерии приёмки (DoD)

- [ ] Для примера плана выполняется 432 полных прогона вместо 25 920.
- [ ] Сумма/склейка пер-слайс сегментов согласована с полной equity-кривой (тест).
- [ ] Warmup на старте истории корректно исключён из метрик.
- [ ] Режим `independent` переключается флагом конфига.
