# AN-3 — Stability score (robust, вектор)

- **Эпик:** E4 — Analytics
- **Статус:** todo
- **Зависимости:** AN-1
- **Реализует улучшение:** №7 (устойчивая метрика стабильности)

## Описание

Надёжная оценка устойчивости параметров вместо хрупкой одиночной формулы.

- `analytics/stability.py`: вектор метрик — `median_return`,
  `positive_slices_ratio`, `worst_slice`, `top_quantile_freq` (с явным определением
  множества: ранг внутри каждого слайса по всем param-комбинациям), `iqr`/дисперсия.
- Защита от деления на ~0 при отсутствии просадки.
- Опциональный композитный score, но основной вывод — вектор + многокритериальное
  ранжирование.

## Критерии приёмки (DoD)

- [ ] На синтетике стабильные параметры (Params B) ранжируются выше «пика» (Params A).
- [ ] Граничные случаи (нулевая просадка, один слайс) не дают NaN/inf.
- [ ] `top_quantile_freq` считается относительно корректного множества (тест).
