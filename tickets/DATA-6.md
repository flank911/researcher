# DATA-6 — Генератор TimeSlice

- **Эпик:** E1 — Data Core
- **Статус:** todo
- **Зависимости:** INF-2
- **Реализует улучшение:** №1 (warmup), частично подготовка к №8/WF (роли)

## Описание

Генерация временных окон как сущностей первого класса.

- `experiments/time_splitters.py`: типы `yearly` / `monthly` / `weekly` / `custom`.
- Каждый `TimeSlice` несёт `warmup_bars`, `min_bars`, `kind`, `label`,
  `role` (`research` / `train` / `test`).
- Фильтрация слайсов по `min_bars` (отбрасывать недостаточно длинные окна).
- Детерминированные `id`/`label` (например `2024-01`, `2024-Q1`).

## Критерии приёмки (DoD)

- [ ] Генерация месяцев 2021–2025 даёт корректные границы и метки.
- [ ] `warmup_bars` корректно прокидывается в каждый слайс.
- [ ] Слайсы короче `min_bars` отфильтровываются.
- [ ] Юнит-тесты (`test_time_splitters.py`), включая граничные даты.
